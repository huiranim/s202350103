## 📌 주요 기능
1) 회원 관리
   * 회원가입 / 탈퇴
   * 로그인 / 아웃
   * 회원정보 변경
   * ID/PW 찾기
   * 마이페이지
     
2) 신규 서적 구매
   * 상품 관리 (등록, 수정, 삭제, 조회)
   * 구매 / 결제
   * 주문 관리 (배송, 취소, 교환, 반품 처리)
   * 선물하기
   * 리뷰
     
3) 중고 서적 구매
   * 매입 및 검수
   * 상품 관리 (등록, 수정, 삭제, 조회)
     
5) 이벤트 및 포인트
   * 이벤트 관리 (등록, 수정, 삭제, 조회)
   * 출석부 / 독서퀴즈 / 친구추천
   * 회원별 포인트 관리
